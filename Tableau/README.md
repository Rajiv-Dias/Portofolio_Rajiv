# Airbnb Dashboard Project (Tableau)

## Overview
This project presents an interactive Tableau dashboard designed to analyze Airbnb listings and pricing patterns across different regions. The dashboard aims to help users understand market distribution, pricing trends, and regional performance using visual analytics.

## Tools
- Tableau Desktop (Public Edition)

## Data
- Source: Airbnb Open Data (public dataset)
- Description: The dataset includes information on listings such as neighbourhood group, room type, price, availability, ratings, and host details.

## Dashboard Features
The dashboard includes several interactive components:
- **Filters**:
  - Room Type
  - Property Type
  - Host Since (Year)
- **Top Region (Bed Availability)**:
  - Compares the availability of beds across neighbourhood groups and room types.
- **Top Region Rating Wise**:
  - Displays average ratings by region to identify areas with higher guest satisfaction.
- **Average Price Based on Zipcode**:
  - A geographic map showing price distribution across zip codes.
- **Average Price per Year**:
  - Line chart illustrating price trends over time.

## Key Insights
- Listing availability is unevenly distributed across regions, with certain areas showing significantly higher bed availability, indicating stronger market concentration.
- Average ratings vary by region, highlighting differences in guest experience and service quality.
- Pricing differs across zip codes, reflecting the influence of location on listing prices.
- The average price trend fluctuates over the years, suggesting changes in market demand and pricing strategies over time.

## Notes
- This dashboard uses publicly available data.
- No sensitive information or credentials are embedded in the workbook.
